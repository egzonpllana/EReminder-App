# EReminder App
Owner of simple and rich reminder that is build with Realm database.

### iOS side of the App ###

- Dependency Injection, Pure Swift Protocol Oriented
- Component based
- Storyboardable
- Services like RealmService or UNUserNotificationService
- MVC Design Pattern
- Fastlane
- Swift 5

### Pods ###
- RealmSwift
- ViewAnimator
- SwiftLint
- CropViewController
- gooey-cell
- liquid-swipe
- Firebase/Analytics
- Firebase/Crashlytics
